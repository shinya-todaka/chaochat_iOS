# chaochat-iOS

## 概要 
reactで開発した[chaochat](https://chaochat.net)をiOSアプリでより使いやすくするためにiOSのクライアントアプリを開発中です。慣れ親しんでいるUIKitではなくSwiftUIで挑戦しています。SwiftUIは難しくコードが汚いのでリファクタリングしながら開発を進めていきたいです。

## 環境
- Xcode 12.3 
- Swift 5.3.2

## 使用技術
- SwiftUI 
- Cloud Firestore
- Cloud Tasks (バックエンド)
- Cloud Functions for Firebase (バックエンド)

## 使用ライブラリ
- Nuke 
- Firebase/Firestore
